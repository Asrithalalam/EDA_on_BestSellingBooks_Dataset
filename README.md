#Best Selling Books – Exploratory Data Analysis (EDA)

##Project Overview

🔸This project performs Exploratory Data Analysis (EDA) on a dataset of the world’s best-selling books.

🔸The goal of this analysis is to:

  - Understand book sales distribution
  
  - Identify top-selling books

  - Extract meaningful business insights using visualizations

🔸The project is implemented using Python, Pandas, Matplotlib, and Seaborn.

##Dataset Description

🔻The dataset contains the following columns:

  - Book, Author, Original language, First published, Approximate sales in millions, Genre 

🔻Total Records: 174 books

##Tools & Libraries Used

◾Python

◾Pandas (data manipulation)

◾Matplotlib (visualization)

◾Seaborn (statistical plotting)

◾OS module (folder management)

##Visualizations Generated

This project generates and saves the following visualizations:

Top 10 Best Selling Books

Sales Distribution

Sales by Genre

Revenue Trends Over Time

All visualizations are automatically saved inside the visuals/ folder as PNG files.

##Key Insights

🔺A small number of books generate extremely high sales.

🔺Certain genres dominate total revenue.

🔺Modern publications show different sales trends compared to older books.

##Conclusion

-This analysis demonstrates how data visualization and aggregation techniques can uncover valuable insights from book sales data.

-The project showcases essential EDA skills such as:

 * Data cleaning
 * Grouping and aggregation
 * Time-series trend analysis
 * Visualization best practices
